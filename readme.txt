It's a WooCommerce extension allowing clients make payment with Umba LoanPay.

== Description ==
Umba LoanPay payment gateway plugin for for WooCommerce which accept payments from client through UmbaLoan pay. Fully supported with WordPress and WooCommerce latest versions. 

= Features =
* Enable and disable Umba LoanPay.
* Enable/Disable Test Mode

== Installation ==
1. Download Umba Payment Gateway from Wordpress Plugin Directory
2. Activate the plugin through the `Plugins` menu in WordPress
3. After activation, you can set options from `WooCommerce -> Settings -> UmbaLoan` menu
4. Enter merchant details
   There are two ways currently to integrate with Umba LoanPay, direct integration or using our partner iPay. Select the relevant option from the dropdown. If you are not sure about this please contact our [support](mailto:engineering@umba.com).
   Based on your integration type you should fill out these fields:
   1. Direct integration
      Merchant Name
      Integration type
      Environment - Umba Stage (testing) or Umba Production (live)
      Umba Security Key - this key needs to be provided by us
      Umba API Key - this key needs to be provided by us
      Vendor ID (provided by us)
   2. Integration with iPay
      Merchant Name
      Integration type
      Select "Make Umba LoanPay Live"
      Vendor ID (assigned by iPay)
      Security Key (assigned by iPay)
      Umba API Key - this key needs to be provided by us
